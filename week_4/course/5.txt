特斯拉（Tesla Inc.）10 月 2 日公布第三季汽車生產數據時，警告中國對該公司開徵的關稅遠多於其他進口車，還聲稱造車成本比直接在當地生產多六成，股價應聲走軟。



特斯拉發表 2018 年第 3 季汽車生產與交車報告時表示，中美貿易關係緊繃，導致中國對特斯拉汽車加徵高達 40% 的進口關稅，遠高於其他進口車的 15%。北京當局會對中國本土製電動車提供相當於建議售價（MSRP）15% 以上的現金補貼，特斯拉卻無法享有這項優惠。

若將海運成本、進口關稅都計入，特斯拉電動車的生產成本，會比直接在中國生產同一輛車貴 55%~60%。由於中國至今仍是全球最大電動車市場，特斯拉面臨的競爭也更具挑戰。為了解決這問題，特斯拉決定加快上海工廠的興建速度。

特斯拉 2 日應聲下挫 3.12% 收 301.02 美元，創 9 月 28 日來收盤低。

特斯拉 2 日新聞稿表示，第 3 季交車量達 83,500 台，其中 55,840 台為 Model 3，14,470 台為 Model S，13,190 台為 Model X，光 Q3 的交車量就達 2017 全年度的 80% 以上。另外，Q3 Model 3 交車量是所有先前季度總和的兩倍之多。CNBC 報導，FactSet 調查顯示，華爾街原本估計，特斯拉 Q3 總交車量僅 80,500 台，Model 3 的交車量為 55,600 台。

特斯拉同時指出，Q3 的汽車生產量為 80,142 台，比 Q2 歷史多出逾 50%，其中 Model S、X 的合併產量為 26,903 台，Model 3 為 53,239 台，幾乎比 Q2 產量多出一倍。另外，Q3 最後一週，Model 3 產量超過 5,300 台，幾乎全都是雙馬達（dual motor）款式。特斯拉先前訂下的 Q3 Model 3 生產目標為 50,000~55,000 台。

根據新聞稿，Q3 底還有 8,048 台 Model 3、3,776 台 Model S 與 Model X 尚在運送，預計 Q4 初就能交到客戶手上。2018 年 Model S 與 Model X 的合併交車目標，仍保持在 100,000 台不變。

CNBC 報導，CFRA 分析師 Garrett Nelson 指出，特斯拉 Q3 產量、交車數據都擊敗華爾街預估，看來營運效率的確有在改善。不過，特斯拉並未提供平均售價或生產成本，令投資人感到失望，部分好消息也早被市場反應完畢。
