位置

我們觀察一下每個位置得到單周最佳球員的次數。

但我們要對這些位置經由一下適當的轉化成五個位置，方便我們分析。 這裡的問題是還沒研究出apply的使用 所以使用迴圈

change_position = function(str_pos){
  if (i =="F-C"){
    return("C")
  }
  if (i =="FC"){
    return("PF")
  }
  if (i =="F-C"){
    return("PF")
  }
  if (i =="F"){
    return("SF")
  }
  if (i =="GF"){
    return("SG")
  }
  if (i =="G-F"){
    return("SG")
  }
  if (i =="G"){
    return("PG")
  }
  return(i)
}


j=0
for( i in data[,7]){
  j = j+1
  data[j,7] = change_position(i)
}

ggplot(data = data, aes(x = Position)) +
  geom_bar() + 
  xlab("pos") + ylab("Total") + 
  ggtitle("Position of players of the week")

整體而言，PG較容易得到當周最佳球員，緊接是小前鋒的位置，但同樣的我們想要看看這些年明星位置的更動。 這裡我們選用一個比較有趣的圖來進行分析，稱violin plot。

ggplot(data = data, aes(x = Position, y = Season.short )) + geom_violin()

這個圖同樣也是很有趣的圖他可以發現各個位置的消長， 1990左右Michael Jordan的時代(“SG占比較高”)到1995左右四大中鋒宰制球場(C較高)。 但2000年以後，有許多傳奇大前鋒(PF)的興起與2010後的明星小前鋒與攻擊型後衛時代(SF與PG較高)
